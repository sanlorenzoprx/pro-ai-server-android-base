# DevStack Offer

Pro Agentic Coding Server turns an old Android phone into a private AI coding assistant for Cursor or VS Code through Continue. The launch offer is local-first and USB-first: the phone runs the local model, the laptop connects to `http://localhost:11434`, and supported local workflows avoid a monthly AI model bill.

Performance depends on the phone. Low-RAM devices should use the lightweight profile and may be slower, especially on first-token response and larger code edits. Do not promise cloud-model speed, large-context reasoning, or perfect autocomplete on older phones.

Trial and refund notes: the free or `$1` entry offer is for compatibility discovery. If the customer's phone cannot pass the checklist, the next step is a better device, not a performance guarantee.

Support boundary: self-serve docs and checklist only.

# Improve Rules

Use this after an implementation report or failure.

Ask:

1. What failed?
2. Why?
3. Could a better rule prevent it?
4. Could a better validation command catch it?
5. Should project memory be updated?
6. Should a command template be improved?

Create mistake record:

```text
.agents/mistakes/{date}-{slug}.md
```

Propose updates to:

```text
.codex/instructions.md
.codex/commands/*.md
.cursor/rules/*.mdc
.agents/workflows/*.md
```

Do not silently update rules. Show proposed changes.

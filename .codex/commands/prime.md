# Prime

Load project context before planning.

Run or inspect:

```bash
git branch --show-current
git status
git log --oneline -n 20
git diff --stat
```

Read:

- README.md
- pyproject.toml
- relevant docs
- relevant source files
- recent reports in `.agents/reports/`
- project memory in `.agents/memory/`

Output:

```text
.agents/memory/last-prime.md
```

Include:

- current branch
- dirty state
- recent commits
- main architecture
- validation commands
- risks
- relevant files

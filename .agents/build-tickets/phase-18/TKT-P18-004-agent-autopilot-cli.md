# TKT-P18-004 Agent Autopilot CLI

## Objective

Expose the controlled autopilot tick through the agent CLI.

## Acceptance Criteria

- `agent autopilot` previews by default.
- `--execute` enables packet writes.
- `--start-session` enables session pickup/start events.
- Phase, ticket, queue, session, ledger, policy, max-ticket, and fail-on-stop options are supported.

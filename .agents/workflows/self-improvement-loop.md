# Self-Improvement Loop

Every coding mistake is a system improvement opportunity.

After implementation or failure, ask:

1. What failed?
2. Why did it fail?
3. Was the plan missing context?
4. Was validation too weak?
5. Was a rule missing?
6. Should a command be improved?
7. Should a test be added?

If yes, create:

```text
.agents/mistakes/{date}-{short-name}.md
```

Then update one of:

```text
.codex/instructions.md
.codex/commands/*.md
.cursor/rules/*.mdc
.agents/workflows/*.md
```

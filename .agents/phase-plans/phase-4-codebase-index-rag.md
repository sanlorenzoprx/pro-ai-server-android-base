# Phase 4: Codebase Index / RAG Foundation

## Goal

Create a dependency-light local codebase index with deterministic chunks, SQLite storage, keyword search, and context building.

## Tickets

- TKT-P4-001: Text chunker and file filtering
- TKT-P4-002: SQLite index store
- TKT-P4-003: Codebase indexer
- TKT-P4-004: Keyword search
- TKT-P4-005: Context builder and CLI commands

## Non-Goals

- No embeddings yet.
- No semantic/vector search yet.
- No background daemon.

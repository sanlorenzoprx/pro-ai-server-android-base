# Mistakes

Store preventable failures here as reusable learning records.

Use this format:

```markdown
# Mistake: {Title}

## Date

{date}

## Context

What was being built?

## Failure

What went wrong?

## Root Cause

- Missing context?
- Bad assumption?
- Weak validation?
- Incorrect implementation pattern?
- Missing test?

## Fix Applied

What changed in code?

## System Improvement

What changed in the AI layer?

## Prevent Next Time

New rule, command, validation, or test to prevent recurrence.
```

# Agent System

This folder stores the project AI layer.

The AI layer is version-controlled context that helps coding agents work consistently in this repository.

It includes:

- PRDs
- implementation plans
- implementation reports
- project memory
- mistake records
- workflow definitions

Core loop:

1. Prime
2. Create phase tickets
3. Define contracts
4. Plan one ticket
5. Implement one ticket
6. Validate
7. Report
8. Improve rules
9. Close phase

# Contract: Launch Tracking

## Phase

Phase 21: DevStack Coding Server Launch

## Purpose

Specify launch attribution before paid traffic, affiliates, or partner outreach.

## Required URL Parameters

- `ref`
- `partner_type`
- `offer`
- `msg`
- `channel`
- `niche`
- `campaign`

## Required Events

- `landing_page_viewed`
- `cta_clicked`
- `lead_captured`
- `trial_started`
- `install_started`
- `install_completed`
- `demo_requested`
- `purchase_completed`

## Validation

- Tracking plan maps each event to required properties.
- Demo and landing docs use the same offer and message tags.

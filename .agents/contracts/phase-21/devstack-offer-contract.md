# Contract: DevStack Offer

## Phase

Phase 21: DevStack Coding Server Launch

## Purpose

Define the first sellable DevStack offer before creator or partner scale.

## Required Offers

- Trial or entry offer around `$1` or free trial.
- Starter offer in the `$49-$99` range.
- Pro install offer in the `$149-$199` range.

## Required Copy Points

- Turn an old Android phone into a private AI coding assistant.
- Works with Cursor or VS Code through Continue.
- Local-first and USB-first.
- No monthly AI model bill for supported local workflows.

## Validation

- Offer doc includes target customer, promise, inclusions, exclusions, and refund/trial notes.
- Pricing does not overpromise performance on low-RAM phones.

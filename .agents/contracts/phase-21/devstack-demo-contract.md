# Contract: DevStack Demo

## Phase

Phase 21: DevStack Coding Server Launch

## Purpose

Create a repeatable demo showing the value of a private local AI coding assistant.

## Demo Must Show

- Android phone connected by USB.
- Pro AI Server status ready.
- Cursor or VS Code using Continue.
- Chat request answered by the local model.
- Autocomplete or code assistance using the local endpoint.
- Positioning line: no monthly AI model bill for supported local workflows.

## Validation

- Demo script has setup, live steps, fallback path, and closing CTA.
- Smoke checklist can be followed by a non-developer operator.
